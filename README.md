This repo contains code used in analyses for the paper The Chronic(les) of Absenteeism Measurement: Unpacking the Many Measures of Attendance and Evidence for a Lower Chronic Absenteeism Threshold. Specifically, it includes code for:
* Creating Figure 1 (Hypothetical Probability of Occurrence Curves) and Figure 2 (Hypothetical ROC Curves)
* Creating ROC plots
* Using the DeLong Test
* Calculating thresholds using Youden's J
